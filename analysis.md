# SBTI 人格测试网站分析文档

## 一、原网站分析

### 1. 视觉设计

**配色方案**
- 主背景色：淡薄荷绿（#e8f0e8 或类似）
- 卡片背景：纯白（#ffffff）
- 主按钮颜色：深绿色（#4d6a53）
- 按钮文字：白色（#ffffff）
- 正文文字：深灰色（#1e2a22 或 rgb(30, 42, 34)）
- 次要文字：灰色

**字体**
- 字体族：-apple-system, BlinkMacSystemFont, "Segoe UI", "PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif
- 标题：加粗，字号较大
- 正文：常规字重

**卡片样式**
- 白色圆角卡片
- 圆角弧度较大
- 轻微阴影效果
- 居中对齐

**按钮样式**
- 深绿色背景
- 白色文字
- 圆角矩形
- hover 效果（颜色变化）

### 2. 页面结构

**首页**
- 标题："MBTI已经过时，SBTI来了。"
- 主按钮："开始测试"
- 底部信息：原作者、B站链接等

**题目页面**
- 顶部：进度条 + 进度文字（"X / 31"）
- 每道题：
  - 题号标签："第X题"
  - 题目内容
  - 三个选项（A/B/C）
  - 单选按钮样式
- 底部：返回首页 + 提交按钮

**结果页面**
- 主类型标签
- 匹配度百分比
- 系统备注
- 人格解读
- 十五维度评分
- 作者的话

### 3. 题目内容特点

- 荒诞、自嘲风格
- 涉及：感情、生活、工作、人际关系
- 选项具有戏剧性和讽刺性
- 符合年轻人网络亚文化

## 二、改进方案

### 左右分栏布局设计

```
+------------------+------------------+------------------+
|                  |                  |                  |
|    上一题        |    当前题目      |    下一题        |
|    (预览)        |    (主区域)      |    (预览)        |
|                  |                  |                  |
|   可点击跳转     |   核心交互区     |   可点击跳转     |
|                  |                  |                  |
+------------------+------------------+------------------+
```

**设计要点**

1. **左侧栏（上一题）**
   - 显示上一题的题目和选项
   - 已选择的答案高亮
   - 可点击跳转到该题

2. **中间栏（当前题目）**
   - 当前正在作答的题目
   - 完整的题目内容和选项
   - 选择后自动切换到下一题

3. **右侧栏（下一题）**
   - 显示下一题的预览（仅题目）
   - 未选择时显示模糊或锁定状态
   - 选择后可预览下一题

### 响应式设计

- 桌面端：三栏布局
- 平板端：三栏布局但宽度调整
- 移动端：单栏布局 + 上下题导航按钮

## 三、技术实现

### 文件结构

```
sbti-clone/
├── index.html      # 主页面
├── style.css       # 样式文件
├── questions.js    # 题目数据
├── app.js          # 交互逻辑
└── analysis.md     # 本分析文档
```

### 核心功能

1. 31道趣味测试题
2. 左右分栏答题界面
3. 选择答案自动切换
4. 进度追踪
5. 生成荒诞人格标签
6. 结果分享功能

### 题目数据结构

```javascript
{
  id: 1,
  question: "题目内容",
  options: [
    { label: "A", text: "选项A内容" },
    { label: "B", text: "选项B内容" },
    { label: "C", text: "选项C内容" }
  ],
  // 用于计算的维度
  dimension: {
    // 不同的选项对应不同的性格维度
  }
}
```

### 结果计算

采用半随机方式：
1. 收集所有答案
2. 根据答案组合计算维度分
3. 结合随机因素生成最终人格
4. 匹配最接近的人格标签
